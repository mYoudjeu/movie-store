## 🎯 À propos

**Movie Store** est une application web complète qui permet de :
- Parcourir une collection de films
- Gérer les informations des films
- Interagir avec une interface utilisateur réactive et moderne
- Stocker et récupérer les données via un backend PHP sécurisé

Ce projet combine un backend robuste en **PHP** avec une interface utilisateur dynamique en **JavaScript** et des styles élégants en **CSS/SCSS**.

## 🚀 Installation

### Prérequis
- **PHP** 7.4+ 
- Un serveur web (Apache, Nginx, etc.)
- Git

### Étapes

1. **Cloner le repository**
   ```bash
   git clone https://github.com/mYoudjeu/movie-store.git
   cd movie-store
   ```

2. **Configurer l'environnement local**
   ```bash
   # Assurer que PHP est accessible
   php --version
   ```

3. **Lancer un serveur local** (exemple avec PHP intégré)
   ```bash
   php -S localhost:8000
   ```

4. **Accéder à l'application**
   - Ouvrez votre navigateur à l'adresse : `http://localhost:8000`

## 💻 Utilisation

1. **Consulter les films** : Parcourez la liste des films disponibles
2. **Ajouter des films** : Remplissez le formulaire pour ajouter un nouveau film
3. **Modifier un film** : Cliquez sur un film pour éditer ses informations
4. **Supprimer un film** : Supprimez un film du catalogue
